# Kotlin 코딩 테스트 연습 (CodingTest_Kotlin)

이 저장소는 Kotlin 언어를 사용하여 코딩 테스트 문제들을 풀이한 기록입니다.

## 📝 프로젝트 설명

IntelliJ IDEA 환경에서 Kotlin을 사용해 다양한 코딩 테스트 문제(주로 백준 온라인 저지 문제로 보임)를 풀이한 코드들을 관리합니다.

## 🛠️ 개발 환경

* **IDE:** IntelliJ IDEA
* **언어:** Kotlin (Language Version: 2.2, API Version: 2.2)
* **JDK:** 17
* **Kotlin 라이브러리:** `org.jetbrains.kotlin:kotlin-stdlib:2.2.20`

## 📂 디렉토리 구조

* `.idea/`: IntelliJ IDEA 프로젝트 설정 파일
* `src/`: 소스 코드 디렉토리
    * `Main.kt`: 기본 메인 파일
    * `seed/`: 문제 풀이 코드가 위치하는 패키지
        * `1000.kt`: A+B
        * `1001.kt`: A-B (실제 파일 내용은 A/B를 %.9f 형식으로 출력)
        * `1330.kt`: 두 수 비교하기
        * `2420.kt`: 사파리월드 (두 수의 차 절댓값)
        * `2753.kt`: 윤년
        * `9498.kt`: 시험 성적
        * `10172.kt`: 개 (출력)
        * `10869.kt`: 사칙연산
        * `11382.kt`: 꼬마 정민 (세 수의 합)
        * `14681.kt`: 사분면 고르기
        * `25083.kt`: 새싹 (출력)
* `.gitignore`: Git 버전 관리에서 제외할 파일 목록
* `CodingTest_Kotlin.iml`: IntelliJ IDEA 모듈 설정 파일
